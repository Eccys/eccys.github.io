# صِيْغَتَا التَّعَجُّبِ

وَهُمَا صِيْغَتَانِ

الْأُوْلَى مَا أَفْعَلَهُ نَحْوُ مَا أَجْمَلَ الْقُرْآنَ 

الثَّانِيَةُ أَفْعِلْ بِهِ نَحْوُ أَجْمِلْ بِالْقُرآنِ
