# الْقِسْمُ الثَّانِيْ فِعْلُ الْمَدْحِ وَالذَّمِّ

وَهِيَ تَرْفَعُ فَاعِلَهَا وَمَخْصُوْصَهَا

نَحْوُ نِعْمَتِ الْعِبَادَةُ الصَّلَاةُ 

وَنَحْوُ بِئْسَ الْخُلُقُ الْكَسَلُ
