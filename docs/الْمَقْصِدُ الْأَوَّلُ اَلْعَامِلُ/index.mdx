> Now we are moving on to the first section, الْمَقْصِدُ الْأَوَّلُ اَلْعَامِلُ. When you see these words, know that another word in the sentence will have an effect because of it. It will cause some change to another word after it or before it, etc.. 


---

The first section of الْمَقْصِدُ الْأَوَّلُ اَلْعَامِلُ is regarding the فعل العامل (the governing verb),

The second section is regarding حرف العامل(the governing particle),

The third is about اسم العامل (the governing noun).

:::info

For example, which أفعال cause the following كلمة to be مرفوع or منصود, which حروف, for example إن, cause the اسم إن to be منصود, and which اسماء causes another كلمة to be مرفوع، مجرور، منصوب،, or مجزوم, etc...

:::

The fourth section is regarding اَلْمَعْنَى (the hidden إعراب, meaning you can't see any عامل, but it is still معمول). 

> For example, المبتدأ. There is nothing before it, but it is still مرفوع.

Lastly, we have اَلْخَاتِمَةُ wherein we discuss some miscellaneous rules which do not fit into any other category.

---
