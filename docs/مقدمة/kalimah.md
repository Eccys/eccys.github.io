# فِي الْكَلِمَةِ

(وَهِيَ قَوْلٌ مُفْرَدٌ),

وَهِيَ ثَلَاثَةُ أَقْسَامٍ:

## الِاسْمُ،

نَحْوُ: الرَّجُلْ،

## الْفِعْلُ


نَحْوُ: عَبَدَ وَ يَعْبُدُ وَ اعْبُدْ

## الْحَرْفُ

نَحْوُ: فِيْ وَ إِنَّ
