# اَلْجُمْلَةُ

وَهِيَ قِسْمَانِ

## الْجُمْلَةُ الِاسْمِيَّةُ

نَحْوُ: اَللَّهُ خَالِقُ كُلِّ شَيْءٍ

## الْجُمْلَةُ الْفِعْلِيَّةُ

نَحْوُ: خَلَقَ اللَّهُ كُلَّ شَيْءٍ
