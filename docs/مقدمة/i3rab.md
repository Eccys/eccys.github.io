# الْعَامِلِ والْمَعْمُوْلِ والْإِعْرَابِ

اَلْعَامِلُ مَا يُوْجِبُ حَالَةً

وَالْمَعْمُوْلُ مَا يَقْبَلُهَ

وَالْإِعْرَابُ أَثَرٌ يَجْلِبُهُ الْعَامِلُ فِيْ آخِرِ الْكَلِمَةِ
