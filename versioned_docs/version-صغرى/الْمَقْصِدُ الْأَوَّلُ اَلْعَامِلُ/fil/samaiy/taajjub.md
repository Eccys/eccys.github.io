# الْقِسْمُ الثَّالِثُ صِيْغَتَا التَّعَجُّبِ

وَهُمَا صِيْغَتَانِ

الْأُوْلَى مَا أَفْعَلَهُ نَحْوُ مَا أَجْمَلَ الْقُرْآنَ
